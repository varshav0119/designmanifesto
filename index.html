Beautiful Websites

  • Material UI Docs
    White Space Plenty of breathing room
    Colors Minimalistic yet good color scheme
    Layout Different components are used well to organize information

  • Notion
    Text Good use of font size to create focal points
    Decorative Elements Emoji used along with text for a unique but minimalistic style

  • HTML5 Up
    Layout Good use of a standard component to display list elements
    Decorative Elements Good use of icons to convey information without using too much text

Best Practices

  • Ensure the the layout is not confusing and use plenty of whitespace between elements - examples from this page are Bootstrap Grid, line breaks and horizontal lines

  • Use consistent text formatting, with different sizes or backgrounds to point out important elements of the page

  • Use a simple and minimalistic color scheme, especially for documentation - in this page, only the default Bootstrap colors are used
